Run Requirements.py first! then Main.py!
Read create_startup.txt on how to start the bot on startup pc (Windows)
The poll.py cog only works with CUSTOM Emoji's.
You ping to bot to use it,
@Bot_Name announce (Message)       > Announces a message using an embeded message (its nice)
@Bot_Name (poll or vote) {} [] [] []			> Message goes inbetween {} and the choices go in between [] one [] per thing to vote. Add a emoji in the [] too so the bot can use that for reactions.
Example:
@bot vote {Hey guys just wanting your opinions, is this bot cool?} [yes] [no] [maybe]
